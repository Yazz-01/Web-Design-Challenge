This is a project made in Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. This was accomplish by utilizing Python libraries, the OpenWeatherMap API, and a little common sense to create a representative model of weather across world cities. Results demonstrates that the atmospheric circulations responds differently to the changes in the ice and snow extents, with a stronger response to sea-ice loss, even though its reduction is half as large as that.
